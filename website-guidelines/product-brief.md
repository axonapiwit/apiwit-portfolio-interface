# Product Brief - APIWIT.EXE Portfolio

## What is it?
Personal portfolio website สำหรับ Apiwit - Frontend Developer
คอนเซปต์ "ระบบปฏิบัติการ .EXE" ธีม cyberpunk dark-red
เว็บจะเปิดขึ้นมาเหมือนบูทโปรแกรม แล้วค่อยเข้าสู่ content หลัก

## Who is it for?

**Primary target users:**
- Recruiters / Hiring managers ที่กำลังหาคน Frontend
- Potential clients ที่ต้องการจ้างงาน freelance
- Fellow developers ที่เข้ามาดูผลงาน

## What's the goal?

As a visitor I want to be able to:
1. เห็นตัวตนและ personality ของ Apiwit ผ่าน visual ที่ unique
2. ดูผลงาน/โปรเจคที่เคยทำ พร้อม tech stack
3. รู้ประสบการณ์การทำงานและ skill set
4. ติดต่อได้ง่ายผ่าน form หรือ social links

## Sections of the website
1. **Loading / Boot Screen** - จำลองการบูทโปรแกรม APIWIT.EXE
2. **Navbar** - minimal, terminal-style navigation
3. **Hero Section** - ชื่อ + role + glitch text effect
4. **About** - bio, avatar, tech stack tags
5. **Projects Showcase** - thumbnail grid + HUD frame
6. **Experience Timeline** - terminal log style
7. **Contact / EOF** - terminal input form + social links

## Tech Stack
- Next.js 16 (App Router)
- React 19
- Tailwind CSS 4
- GSAP (animations: glitch, scroll reveal, typing, marquee)
- TypeScript
