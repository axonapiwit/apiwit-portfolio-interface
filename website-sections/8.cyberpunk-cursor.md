# Cyberpunk Custom Cursor

## Concept
ซ่อน cursor ปกติ แล้วแทนด้วย custom cursor สไตล์ cyberpunk HUD crosshair
มี 2 ส่วน: outer ring (ตามช้า) + inner dot (ตามเร็ว) สร้าง parallax feel
เปลี่ยนรูปร่างตาม context (hover link, hover card, text, loading)

## Structure
```
<div id="cursor" class="pointer-events-none fixed inset-0 z-[9999]">
  <div class="cursor-outer" />   <!-- outer ring / crosshair -->
  <div class="cursor-dot" />     <!-- inner dot -->
</div>
```

## Cursor States

### Default (idle/move)
- outer: 36x36px, border 1px accent red/50, rounded-full
- inner: 6x6px, bg accent red, rounded-full
- outer ตาม mouse ด้วย lerp (delay ~0.15s) → สร้าง smooth lag
- inner ตาม mouse ทันที (no delay)

### Hover Link / Button (`data-cursor="link"`)
- outer: scale up → 48x48px, border accent red, opacity เพิ่ม
- inner: scale down → 4px
- mix-blend-mode: difference (optional)
- transition: 300ms ease

### Hover Project Card (`data-cursor="project"`)
- outer: เปลี่ยนเป็นรูป crosshair/HUD bracket `[ + ]`
  - 4 เส้น L-shape มุม (เหมือน camera viewfinder)
  - ขนาด 56x56px
  - สีแดง accent
- inner: คงเดิม
- optional: text label เล็ก ๆ `VIEW` ใต้ crosshair

### Text Area (`data-cursor="text"`)
- outer: ซ่อน
- inner: เปลี่ยนเป็น vertical bar `|` สีแดง, 20px สูง, กะพริบ

### Loading / Disabled
- outer: spinning ring animation (rotate 360 loop)
- inner: pulse

## Glow Trail (optional)
- ทุก frame จด position → วาดจุด trail 5-8 จุด
- แต่ละจุด opacity ลดลง + ขนาดเล็กลง
- สีแดง accent glow
- ใช้ canvas overlay หรือ DOM elements recycle

## Implementation

### Hook: `useCyberpunkCursor()`
```
- mouseX, mouseY: track ด้วย mousemove event
- outerX, outerY: lerp ตาม mouse (GSAP quickTo หรือ requestAnimationFrame)
- cursorState: "default" | "link" | "project" | "text" | "loading"
- detect hover target ด้วย data-cursor attribute
- ใช้ GSAP .to() สำหรับ state transition
```

### CSS
```css
html, body, a, button {
  cursor: none !important;
}
```

### Performance
- ใช้ `transform: translate3d()` (GPU accelerated)
- ไม่ใช้ top/left (จะ trigger layout)
- GSAP `quickTo()` สำหรับ lerp ที่ smooth
- ใช้ `will-change: transform` บน cursor elements

## Responsive
- **Desktop only**: แสดง custom cursor เฉพาะ device ที่มี mouse
- **Touch devices**: ซ่อน custom cursor ทั้งหมด
- detect: `@media (hover: hover) and (pointer: fine)`
- หรือ JS: `window.matchMedia('(hover: hover)')` 

## Accessibility
- `prefers-reduced-motion`: ปิด trail + lerp delay, cursor ตามทันที
- cursor elements มี `aria-hidden="true"`
- ไม่กระทบ tab navigation (pointer-events: none)
