## Project Card

### Structure
```
<div class="project-card">
  <div class="card-thumbnail">
    <img />
    <div class="card-overlay" />      <!-- hover overlay -->
    <div class="card-corners" />       <!-- HUD corner marks -->
  </div>
  <div class="card-content">
    <h3 class="card-title">Project Name</h3>
    <p class="card-description">Short description...</p>
    <div class="card-tags">[React] [Next.js]</div>
    <div class="card-links">
      <CyberButton variant="primary" size="sm">LIVE</CyberButton>
      <CyberButton variant="ghost" size="sm">CODE</CyberButton>
    </div>
  </div>
</div>
```

### Styling
- bg: bg-secondary (#111)
- border: border-white/8
- hover: border-accent-red/30 + shadow accent-glow
- transition: all 300ms
- thumbnail: aspect-video, object-cover, grayscale → hover: color
- content padding: p-6

### HUD Corner Marks (Pseudo Elements)
- 4 มุมของ card มี L-shape accent marks
- สีแดง accent, ขนาด ~12px per side
- ใช้ `::before` / `::after` หรือ border trick

### Hover Effect
- thumbnail: grayscale(100%) → grayscale(0%)
- border: white/8 → accent-red/30
- optional: subtle CSS glitch filter
