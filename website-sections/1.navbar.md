# Navbar - APIWIT.EXE

## Layout
- fixed top, full width, z-50
- height: 64px (desktop), 56px (mobile)
- padding: `px-6 md:px-12`
- background: transparent → `bg-black/80 backdrop-blur-md` on scroll (>50px)
- border-bottom: `border-b border-white/8` on scroll

## Structure
```
[APIWIT.EXE]                    [> ABOUT] [> PROJECTS] [> EXPERIENCE] [> CONTACT]  (desktop)
[APIWIT.EXE]                    [☰]                                                (mobile)
```

## Logo (Left)
- text: `APIWIT.EXE`
- font: Poppins 600, 16px, tracking-wider, uppercase
- color: text-primary (#ededed)
- hover: text-accent (#ff2d2d)
- click: scroll to top

## Nav Items (Right - Desktop)
- prefix `>` ก่อนแต่ละ item (terminal style)
- font: Poppins 500, 14px, tracking-wide, uppercase
- color: text-secondary (#888) → hover: text-accent (#ff2d2d)
- active (current section): text-accent + underline offset
- gap ระหว่าง items: `gap-8`
- transition: color 200ms

## Mobile Menu
- trigger: hamburger icon (3 horizontal lines) → X เมื่อเปิด
- overlay: full screen, bg-black/95, backdrop-blur
- items แสดง vertical center, font ใหญ่ขึ้น (24px)
- stagger animation เข้า: GSAP fade + slideX จากขวา
- ปิดเมื่อ: กด X, กด item, กด Escape

## Scroll Behavior
- ใช้ IntersectionObserver track section ที่อยู่ใน viewport
- highlight nav item ที่ตรงกับ section ปัจจุบัน
- smooth scroll เมื่อกด nav item

## Animation
- mount: fade in จาก opacity 0 → 1 (delay หลัง boot screen)
- scroll bg transition: CSS transition 300ms
- mobile menu: GSAP timeline stagger
