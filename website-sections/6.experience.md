# Experience Section

## Layout
- section id: `experience`
- max-w-4xl mx-auto
- padding: `py-24 md:py-32 px-6`

## Section Header
- label: `> EXPERIENCE`
- title: `System Log`
- same header style

## Timeline

### Structure
- vertical line ซ้าย: `border-l border-white/10` (desktop) หรือ `border-l` (mobile)
- แต่ละ entry อยู่ขวาของ line
- dot indicator บน line: `w-3 h-3 bg-accent rounded-full`

### Entry Layout
```
○ ─── [2024] > Senior Frontend Developer @ Company
       Description of role and responsibilities.
       Tags: [React] [Next.js] [TypeScript]
```

### Entry Styling
- year: monospace, text-accent, font-bold
- role + company: Poppins 600, 18px, text-primary
- description: Poppins 400, 14px, text-secondary, max 2 lines
- tags: same tag style as About section
- gap between entries: `gap-12`

### Terminal Log Variant (alternative)
แสดงเป็น terminal output:
```
[2024.01] > EXEC role="Senior Frontend" --at="Company Name"
           status: ACTIVE
           stack: [React, Next.js, TypeScript]

[2022.06] > EXEC role="Frontend Developer" --at="Company Name"
           status: COMPLETED
           stack: [Vue, Nuxt, SCSS]
```

## Animation
- scroll reveal: timeline entries stagger from left
- dot indicator: scale pulse เมื่อ enter viewport
- line: draw from top to bottom ตาม scroll (GSAP drawSVG style)

## Data Schema
```ts
type Experience = {
  id: string
  year: string
  role: string
  company: string
  description: string
  tags: string[]
  status: "active" | "completed"
}
```
