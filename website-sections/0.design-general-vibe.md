# Design General Vibe - APIWIT.EXE

## Concept
"ระบบปฏิบัติการที่บูทขึ้นมาแล้วโชว์ตัวตน"
Cyberpunk terminal aesthetic - ดำเข้ม แดงเป็น accent, ดิบ เหลี่ยม ไม่ rounded
ทุกอย่างรู้สึกเหมือนกำลังอยู่ใน interface ของโปรแกรมอะไรสักอย่าง

## Mood Keywords
- Dark, moody, high contrast
- Terminal / HUD / System UI
- Glitch, distortion, digital noise
- Bold typography, sharp edges
- Red accent on black canvas

## Visual References
- Ray-Ban.EXE campaign (inspire/1-4.png)
- Boot screen → loading bar → reveal content
- HUD frame borders รอบ cards
- Marquee ticker text แบบ broadcast
- Scramble/decode text animation

## Do's
- ใช้ monospace font สำหรับ decorative text (marquee, labels)
- ใช้ border accent แดงเฉพาะจุด ไม่ใช้เยอะเกิน
- ให้ negative space เยอะ ไม่แน่นเกิน
- animation ต้อง subtle ไม่รกตา (ยกเว้น hero glitch)
- ทุก section ควรมี visual hierarchy ชัดเจน

## Don'ts
- ห้าม rounded corners มาก (max rounded-sm)
- ห้ามใช้สีสดใสอื่นนอกจาก red accent
- ห้าม animation หนักทุกที่ - เลือกใช้เฉพาะจุด focus
- ห้ามใช้ stock photo ที่ดูไม่เข้ากับ theme
- ห้าม white background ที่ไหนเลย
